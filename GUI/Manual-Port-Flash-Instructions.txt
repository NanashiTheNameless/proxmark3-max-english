Instructions for manually specifying a port to flash firmware:

Some Windows 10/11 systems have issues where PM3 cannot automatically select the port.
If you encounter this problem, follow these steps to manually specify the port for flashing.

Step 1: Click "I can't brush the point where the firmware can't connect.bat"
  This batch file copies the contents of the libs folder inside the Client folder to the Client root directory.
  If the batch file does not copy automatically, copy the files manually — otherwise Steps 2 and 3 cannot run.

Step 2: After copying, click "Manual-Flash-BOOT.bat"

Step 3: After BOOT flashing is complete, click "Manual-Flash-FULLIMAGE.bat"
